## LCD / LVGL

### 几何、颜色和底层参数

- LVGL 逻辑屏幕：`320x172`，坐标原点左上角，`x=0..319`，`y=0..171`
- LCD 控制器：JD9853，SPI，16-bit RGB565；驱动配置为 BGR 顺序，flush 前会做 RGB565 字节交换
- `lv_displayer` 创建 `lv_display_create(320, 172)`，使用双 DMA 缓冲，每个缓冲为 `320*20` 像素，`LV_DISPLAY_RENDER_MODE_PARTIAL`
- mPython Pro 板载 LCD 引脚：MOSI 37，SCK 36，CS 34，DC 35，RST -1，BL 33，SPI 像素时钟 40MHz
- 面板方向参数：`swap_xy=True`，`mirror_x=True`，`mirror_y=True`，`gap_x=0`，`gap_y=34`，`invert=False`
- 直接用底层 `lcd` 模块时只有 `lcd.init()`、`lcd.deinit()`、`lcd.draw_logo()`、`lcd.draw_color(color)` 等很薄的接口；一般 UI 用 LVGL。

### 反相显示

- 固件配置里 `BOARD_LCD_INVERT` 当前为 `false`；初始化时 `lcd_init()` 调用 `esp_lcd_panel_invert_color(lcd->panel, BOARD_LCD_INVERT)`。
- 这是 JD9853 面板控制器层面的全屏颜色反相，底层发送 `LCD_CMD_INVON` / `LCD_CMD_INVOFF`，通常用于适配面板电气/色彩方向，不是 LVGL 主题样式。
- 若固件把 `BOARD_LCD_INVERT` 改成 `true`，所有 RGB565 输出都会被面板反相，包括图片、背景和控件颜色。
- Python/LVGL 常规 UI 不要依赖面板反相做局部效果；菜单选中态、按钮按下态、局部反色应通过 LVGL 样式设置前景色/背景色实现。

**重要**：使用 LVGL 前必须先 `import lv_displayer`（会自动初始化 display）：

```python
import lv_displayer  # 自动调用 __init__，创建 display 对象
import lvgl as lv
from lv_utils import event_loop

el = event_loop(freq=25)  # 默认 25Hz 刷新

scr = lv.screen_active()  # LVGL 9.3 API
label = lv.label(scr)
label.set_text("Hello HandPy")
label.align(lv.ALIGN.CENTER, 0, 0)
```

**lv_gui.py 封装**：预置的 `lv_gui.py` 已自动 import `lv_displayer`，直接使用即可。提供高级封装（二维码/多边形/位图/按键映射），详见 `./modules/lv_gui.md`。

### 内置字体与文字尺寸

实际启用字体以固件 `lv_conf.h` 为准：

| 字体对象 | 用途 | line_height / baseline | 字符宽度要点 |
|----------|------|------------------------|--------------|
| `lv.font_montserrat_16` | 默认字体，ASCII/数字/符号 | 18 / 3 | `A` 约 11.7px，`0` 约 10.7px，`W` 18px，`i` 4.4px |
| `lv.font_montserrat_20` | 较大 ASCII/数字/符号 | 22 / 4 | `A` 约 14.6px，`0` 约 13.3px，`W` 22.5px |
| `lv.font_montserrat_24` | 标题/大数字 | 27 / 5 | `A` 约 17.6px，`0` 16px，`W` 27px |
| `lv.font_siyuan_heiti_medium_24` | 中文/CJK | 45 / 13 | 常见中文 advance 24px，字形盒约 18~22x22px |

要点：

- 默认字体是 `lv.font_montserrat_16`，不覆盖中文；显示中文时显式设置 `lv.font_siyuan_heiti_medium_24`。
- `lv_font_simsun_14_cjk.c` 和 `lv_font_simsun_16_cjk.c` 源码存在，但当前 `lv_conf.h` 未启用，不要默认使用 `lv.font_simsun_14_cjk` / `lv.font_simsun_16_cjk`，除非确认固件重新打开了这些宏。
- LVGL 字体宽度是 proportional，不能把“字号”直接当成每字宽。思源黑体 24 下中文通常可按 `24px` 推进估算：320px 一行约 13 个中文；line_height 45px 下 172px 高度约 3 行。
- 文字大小受字体、`letter_space`、`line_space`、label 宽度和换行策略共同影响；做精确布局时让 LVGL 布局后读 label 尺寸，而不是手算。

```python
label = lv.label(scr)
label.set_text("中文")
label.set_style_text_font(lv.font_siyuan_heiti_medium_24, 0)
label.align(lv.ALIGN.CENTER, 0, 0)

# 需要测量时，等布局更新后读实际对象尺寸
label.update_layout()
print(label.get_width(), label.get_height())
```

### 性能优化技巧

#### 局部刷新

LVGL 默认支持局部刷新（dirty area tracking），但需要正确使用：

```python
import lv_displayer
import lvgl as lv
from lv_utils import event_loop

el = event_loop(freq=25)

scr = lv.screen_active()
label = lv.label(scr)
label.set_text("Count: 0")
label.align(lv.ALIGN.CENTER, 0, 0)

# 更新时只触发局部刷新
count = 0
while True:
    count += 1
    label.set_text(f"Count: {count}")  # LVGL 自动标记 dirty area
    # 不需要手动调用 invalidate()，set_text 会自动触发
    time.sleep(0.1)
```

**手动触发局部刷新：**
```python
# 只在需要强制刷新时使用
label.invalidate()  # 标记该对象需要重绘
```

#### event_loop 频率调优

默认 25Hz（40ms/帧）适合大多数场景。根据需求调整：

```python
# 低功耗场景（静态显示）
el = event_loop(freq=10)  # 100ms/帧

# 流畅动画场景
el = event_loop(freq=50)  # 20ms/帧

# 极致性能（注意 CPU 占用）
el = event_loop(freq=60)  # 16.7ms/帧
```

**权衡：**
- 频率越高 → 动画越流畅，但 CPU 占用越高、功耗越大
- 频率越低 → 省电，但可能出现卡顿

#### 避免全屏刷新

❌ **错误做法：**
```python
# 每次都清空整个屏幕
scr.clean()
# 重新绘制所有元素
label1.set_text("...")
label2.set_text("...")
# 触发全屏刷新
```

✅ **正确做法：**
```python
# 只更新变化的元素
label1.set_text("new value")  # LVGL 自动局部刷新
```

#### 内存优化

v3 有 PSRAM（8MB+），但大对象（>512B）会分配到较慢的 PSRAM。

**优化建议：**
- 小对象（<512B）自动分配到快速 SRAM
- 避免频繁创建/销毁大对象（预分配）
- 图片资源优先用内置的 images/ 目录（已在 Flash）

**监控内存：**
```python
import gc
print(f"Free: {gc.mem_free()}, Alloc: {gc.mem_alloc()}")
gc.collect()  # 主动回收
```

详细性能优化技巧见 `patterns.md`。
